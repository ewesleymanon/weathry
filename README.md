# weathry

## Installation

1. Install the dependencies.
```
npm install
```

3. Env
```
cp .env.sample .env
```

2. Start the FE server. 

**NOTE:** 
#Docker
```
docker compose up
```
Local
```
npm start
```

3. Done! The FE server should now be running and `http://localhost:3000` should be open in your browser.
